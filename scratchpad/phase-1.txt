Step-by-step Instructions (Copy-Paste One Command at a Time)
1. Update your system and install required tools
Open your terminal and run:
Bashsudo apt update && sudo apt upgrade -y
sudo apt install curl git -y
2. Install Node.js (Latest LTS — currently v20 or v22 is fine)
Bashcurl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
sudo apt install -y nodejs
node --version   # Should show v20.x or v22.x
npm --version
3. Install MongoDB Community Edition (Local)
Bash# Import MongoDB public GPG key
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg --dearmor

# Add MongoDB repository (for Ubuntu 22.04 or 24.04)
echo "deb [ arch=amd64,arm64 signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] https://repo.mongodb.org/apt/ubuntu $(lsb_release -cs)/mongodb-org/8.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list

sudo apt update
sudo apt install -y mongodb-org

# Start and enable MongoDB
sudo systemctl start mongod
sudo systemctl enable mongod

# Check status
sudo systemctl status mongod   # Should show "active (running)"
If you see any error about port or already installed, just run sudo systemctl restart mongod.
4. Create Project Folder & Initialize Git
Bashmkdir ~/magnus-more-clone
cd ~/magnus-more-clone
git init
5. Create Backend and Frontend Folders + Initialize
Bash# Create folders
mkdir backend frontend

# Backend setup
cd backend
npm init -y
npm install express mongoose cors dotenv multer nodemon
npm install --save-dev

# Update backend package.json scripts (we'll do it manually below)
cd ..

# Frontend setup (Vite + React)
cd frontend
npm create vite@latest . -- --template react
npm install
npm install react-router-dom axios
cd ..
6. Set up basic folder structure
Run these commands from the root (~/magnus-more-clone):
Bash# Backend folders
mkdir -p backend/{config,controllers,models,routes,middleware,utils}

# Frontend folders (important ones)
mkdir -p frontend/src/{components/pages/More,components/common,services,context,assets}

# Create basic files
touch backend/.env
touch backend/server.js
touch frontend/vite.config.js
7. Configure .env (Backend)
Open backend/.env with nano or any editor and paste:
envPORT=5000
MONGO_URI=mongodb://127.0.0.1:27017/magnusdb
NODE_ENV=development
8. Basic server.js (Backend)
Create/edit backend/server.js:
JavaScriptconst express = require('express');
const cors = require('cors');
const dotenv = require('dotenv');
const mongoose = require('mongoose');

dotenv.config();

const app = express();
const PORT = process.env.PORT || 5000;

// Middleware
app.use(cors());
app.use(express.json());
app.use(express.urlencoded({ extended: true }));

// Test route
app.get('/api/health', (req, res) => {
  res.json({ message: 'Magnus Backend is running successfully!' });
});

// Connect to MongoDB
mongoose.connect(process.env.MONGO_URI)
  .then(() => console.log('✅ MongoDB Connected'))
  .catch(err => console.error('❌ MongoDB Connection Error:', err));

app.listen(PORT, () => {
  console.log(`🚀 Server running on http://localhost:${PORT}`);
});
Update backend/package.json scripts section to:
JSON"scripts": {
  "start": "node server.js",
  "dev": "nodemon server.js"
}
9. Frontend Proxy Setup (to avoid CORS in development)
Edit frontend/vite.config.js:
JavaScriptimport { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [react()],
  server: {
    port: 3000,
    proxy: {
      '/api': {
        target: 'http://localhost:5000',
        changeOrigin: true,
        secure: false,
      }
    }
  }
})
10. Commit this stage to Git
From root folder:
Bashgit add .
git commit -m "Phase 1: Initial project setup - Backend + Frontend + MongoDB + Proxy"
How to Run the Project (Test Now)
Open two terminals:
Terminal 1 (Backend):
Bashcd ~/magnus-more-clone/backend
npm run dev
Terminal 2 (Frontend):
Bashcd ~/magnus-more-clone/frontend
npm run dev

Backend should show: "MongoDB Connected" and "Server running on http://localhost:5000"
Frontend should open at http://localhost:3000 (default Vite page for now)

Go to http://localhost:5000/api/health → You should see the JSON message
